# DSA-Road-Map

🧠 60-Day DSA Roadmap (Deep + Safer)

👉 Best if you're starting from scratch or targeting top companies.

🗓️ Days 1–10: Basics + Arrays
Arrays, strings
Sliding window, two pointers
Solve ~30 problems
🗓️ Days 11–20: Hashing + Stack + Queue
HashMaps deeply
Stack patterns (monotonic)
Queue + BFS
Solve ~35 problems
🗓️ Days 21–30: Linked List + Recursion
Linked list patterns
Recursion fundamentals
Backtracking intro
🗓️ Days 31–40: Trees (Very Important)
DFS, BFS
Tree patterns (diameter, LCA)
BST
Solve ~40 problems
🗓️ Days 41–45: Graphs
BFS, DFS
Cycle detection
Topological sort
Shortest path basics
🗓️ Days 46–55: Dynamic Programming
1D DP (Fibonacci, climbing stairs)
2D DP (grid, LCS)
Knapsack pattern
🗓️ Days 56–60: Advanced + Revision
Heap / Priority Queue
Greedy
Binary Search advanced
Mock interviews daily
